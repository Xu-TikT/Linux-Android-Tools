Please note that there are multiple servers working behind an individual DNS. This means that when you use one DNS address, it automatically connects you to one of the fastest servers available behind that particular DNS address.
If one server is down for maintenance, you will automatically connect to the next available server.

How to use?

1- Use the below search bar to find your desired DNS address as per your desired country/city.
