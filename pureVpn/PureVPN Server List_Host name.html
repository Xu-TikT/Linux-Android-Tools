We have compiled the list of Domain Naming Server (DNS) addresses for PPTP, L2TP, IKEv2, SSTP, OpenVPN-UDP and OpenVPN-TCP respectively. You can use any DNS address with your selected protocol. Please note that there are multiple servers working behind an individual DNS. This means that when you use one DNS address, it automatically connects you to one of the fastest servers available behind that particular DNS address.
If one server is down for maintenance, you will automatically connect to the next available server.

How to use?

1- You can use the search bar below to find your desired DNS address as per your desired country/city.

2- Once you have found your desired address, simply copy and paste it in PureVPN’s software dedicated IP box Or in the server location field, in case you are configuring PureVPN manually.
