                                                      Hi,

                                                      I am sorry for the inconvenience. Allow me explain you might be facing this issue:
                                                      If you visit a website before connecting to VPN, it records your location then if you re-visit the same website after connecting to VPN, it still shows your current location because that location is already recorded in website history.
                                                      Possible Solutions:
                                                      - Delete your browser cookies, re-connect VPN and visit the site again
                                                      - If that does not work then open your browser in Incognito/Private mode
                                                      - If problem persist, restart your system and try again
